# Frontend Architecture (React)

Project: Acme Portal
Version: 1.0
Status: Active

## Structure

- `src/pages` for route-level screens
- `src/components` for reusable UI
- `src/features/requests` for domain feature modules

## Data strategy

- Server state fetched from Flask API.
- Client state scoped to interaction and forms.
