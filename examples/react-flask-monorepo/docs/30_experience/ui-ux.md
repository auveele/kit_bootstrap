# UI UX Spec

Project: Acme Portal
Version: 1.0
Status: Active

## UX objective

Enable fast request processing with low cognitive load.

## Critical flows

1. Sign in and access dashboard.
2. Create and edit request.
3. Assign request and track status changes.

## Responsive behavior

- Desktop first with tablet support.
- Mobile focuses on status updates and comments.
