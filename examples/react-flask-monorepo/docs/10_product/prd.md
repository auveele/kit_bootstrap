# PRD

Project: Acme Portal
Version: 1.0
Status: Active

## Vision

Deliver a web portal where operations teams can manage customer requests,
track status, and collaborate across departments.

## MVP scope

In:
- user authentication
- request creation and status tracking
- team assignment workflow

Out:
- advanced analytics dashboards
- external marketplace integrations

## KPI and thresholds

| KPI | Target |
|---|---|
| Request completion time | < 48h median |
| Weekly active users | >= 60 |
