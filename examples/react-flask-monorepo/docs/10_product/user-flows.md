Project: Acme Portal
Version: 1.0

## Critical journeys

1. User signs in and lands on request board.
2. User creates a request and submits details.
3. Team lead assigns owner and updates status to done.

## Flow acceptance signals

- no blocking error in primary flow
- success message for create/update actions
