# React Flask Monorepo Example

This example shows a generated `docs/` execution system for a React + Flask
project using SQLAlchemy and SQLite bootstrap.

It also includes the mandatory root layout baseline:
- `.run_cache/`
- `backend/.venv/`
- `docs/`
- `frontend/`
- `instance/`
- `scripts/`
- `tools/`
- `.gitignore`
- `README.md`

Run checks:

```bash
python tools/bootstrap.py check --root examples/react-flask-monorepo/docs
```
